COLLECTION OF YOUR INFORMATION

The application uses Google OAuth feature to request permission for accessing your workout informations that’s the application does not collect and does not share with us or any 3rd party person or company.

 

Personal Data

The application can download the user’s workout data that only the user’s phone and watch have access to through third party services like Google Fitness API. The data on the phone will be deleted after the request is fulfilled and the information is sent to the user’s watch via the Wear OS’s Wearable Service. The watch shows the data on the watchface until there is new data to show, or the user deletes the application storage of the watch.

 

Derivative Data 

The Google Cloud Platform may automatically collect usage information of the application for creating statistics of the newly registered users, how many users using the application or the number of the requests.

 

Google Cloud Platform

The application uses Google Cloud Platform to have access to the Google Fit API and to allow the application to have Google OAuth feature.

 

 

Google Fit API

The Google Fit API uses the Google Play Services on the user’s phone to get access to the workout information.

 

Mobile Device Access

We may request access or permission to certain features from your mobile device, including your mobile device’s Wi-Fi, sensors and other features. If you wish to change our access or permissions, you may do so in your device’s settings.

 

Mobile Device Data

Device information such as your mobile device ID number, model, and manufacturer, version of your operating system, phone number, country, location, and any other data you choose to provide.

 

Third-Party Data
